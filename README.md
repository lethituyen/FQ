# FuckTomato

`FuckTomato` 是一个面向番茄免费小说 `com.dragon.read` 的 Xposed 模块，目标是减少干扰内容、精简界面并提升阅读体验。

## 功能介绍

- 去广告与去福利
  - 禁用广告配置与免广告相关判断
  - 去除 Banner、LuckyDog 福利等广告内容
  - 拦截常见弹窗与点击统计

- 阅读体验优化
  - 禁用作者说、封面热评、章末热评
  - 去除章末评论开关、礼物等干扰控件
  - 阅读页返回时直接结束当前页面

- 页面精简
  - 精简首页顶部频道，仅保留常用分类
  - 隐藏底部 Tab 与排行榜
  - 清空搜索热词，过滤搜索页部分推荐内容
  - 精简“我的”页面功能卡片，移除多余入口与红点提示
  - 禁用“我的”页侧边栏，回退为设置按钮模式
  - 隐藏“我的”页动态搜索入口，并保留顶部按钮占位避免错位

- VIP 相关处理
  - 隐藏“我的”页面 VIP 入口
  - 修改部分 VIP 信息展示

- 系统行为拦截
  - 禁用 ABTest
  - 拦截更新消息与更新检查
  - 屏蔽部分埋点、上报、崩溃监控与相关后台调度

- 下载导出
  - 捕获已下载章节正文
  - 下载完成后自动整理并导出 TXT 文件
